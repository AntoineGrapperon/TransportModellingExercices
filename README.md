# Transport Modelling Exercises

A Streamlit-based educational application designed to teach and practice transport modelling theoretical concepts and exercises.

## Project Goal
The aim of this project is to provide an interactive platform where students and professionals can learn the fundamentals of transport modelling and solve practical exercises.

## Current Focus
We are currently developing the modules for the **4-Step Demand Model**:
1. **Trip Generation**: Estimating the total number of trips produced and attracted in each zone.
2. **Trip Distribution**: Distributing trips between zones (e.g., using Gravity Models).
3. **Modal Split**: Determining the proportion of trips made by different transport modes (e.g., Logit Models).
4. **Traffic Assignment**: Assigning trips to the physical transport network (e.g., User Equilibrium, All-or-Nothing).

## Tech Stack
- **Python**: Core logic.
- **Streamlit**: Web interface and interactivity.
- **NumPy & Pandas**: Data manipulation and numerical computations.
- **Matplotlib/Plotly**: Visualization.

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`
